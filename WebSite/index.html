Introduction

1) The main goal of the project is the management of the university database. The aim of this project is to develop software for the effective management of a university. This DBMS will be used by University's faculties. Staffs can use this DBMS for maneging, accessing and updating infomation about students, proforessors, courses, libraries and so on.

2) The System can find, update, modify and delete information from the university database. For example: We can easily find infomation about what course student is in or what their grad is. Also which professor teaches which course. In additions we can access data on library, assistants, departments, sections, courses etc.
